# webpack-simple 

> A simple Vue 2.0 Webpack & `vue-loader` setup for quick prototyping. Note this template is not suitable for production - for that you may want to wait for an official 2.0 webpack template.

> 一个简单的Vue 2.0 Webpack 和 vue-loader 快速原型设置。请注意 ，此模板不适合生产 - 因此你可能希望等待官方的2.0 Webpack模板 

> This template is Vue 2.0 compatible. For Vue 1.x use this command: `vue init webpack-simple#1.0 my-project`

> 此模板与Vue 2.0 兼容。对于Vue 1.x 请使用命令：`vue init webpack-simple#1.0 my-project`

### Usage | 用法

This is a project template for [vue-cli](https://github.com/vuejs/vue-cli).
这是一个 [vue-cli](https://github.com/vuejs/vue-cli) 的项目模板

``` bash 
$ npm install -g vue-cli
$ vue init webpack-simple my-project
$ cd my-project
$ npm install
$ npm run dev
```

### What's Included | 包括什么？ 
- `npm run dev`: Webpack + `vue-loader` with proper config for source maps & hot-reload.
- `npm run dev`: Webpack + `vue-loader` 带有资源映射 和 热重载的适当配置

- `npm run build`: build with HTML/CSS/JS minificaton.
- `npm run build`: 使用 HTML/CSS/JS 最小化构建

For detailed explanation on how things work, consult th [docs for vue-loader](http://vuejs.github.io/vue-loader). Also check out the [breaking changes in vue-loader@9.0.0](https://github.com/vuejs/vue-loader/releases/tag/v9.0.0).
对于如何工作的详细说明，请参考[vue-loader的文档](http://vuejs.github.io/vue-loader) ，也请检查vue-loader@9.0.0中的[重大变化](https://github.com/vuejs/vue-loader/releases/tag/v9.0.0)

### Fork It And Make Your Own | 检出作为自己的项目

You can fork this repo to create your own boilerplate, and use it with `vue-cli`:
你可以使用 vue-cli 检出这个仓库，以创建你自己的样版

``` bash
vue init username/repo my-project
```
